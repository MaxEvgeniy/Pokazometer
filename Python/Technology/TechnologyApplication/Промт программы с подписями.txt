Нужна программа на Python, которая:
1. Открывает документ .pdf из папки по адресу "C:\PDF\1\Rename".
2. Определяет формат документа. Документ может быть формата А4 или А3. 
3. Вписывает слово "Утвердил" в указанное место с заранее определенными координатами.
3. Вписывает слово "Проверил" в указанное место с заранее определенными координатами.
4. Вставляет картинку из шаблона "Утвердил.tif" в указанное место с заранее определенными координатами.
4. Вставляет картинку из шаблона "Проверил.tif" в указанное место с заранее определенными координатами.
5. Считывает значение даты из места с определенными заранее координатами. В случае ошибки считывания, за значение даты принимается текущий день на системном календаре за вычетом 14 дней.
6. Считанное значение даты увеличиваем на срок, выбранный случайным образом от 3 до 14 дней.
7. Вставляем получившееся значение даты в указанное место с заранее определенными координатами.
8. Сохраняем и закрываем получившийся документ .pdf.
9. Повторяем все для каждого .pdf файла из той же папки.

Координаты каждого места для редактирования объявлены константами, чтобы я смог их скорректировать.
Вокруг каждой редактируемой или считываемой области нарисовать красную рамку, чтобы удобнее было подобрать координаты.
В итоговом варианте программы рисование этих рамок будет закомментировано. 
Местоположение областей для формата А3 и А4 различно. Координаты областей для А3 высчитываются из координат для области А4 при помощи коэффициентов, которые должны быть предварительно объявлены константами.
Вставляемые картинки имеют прозрачный фон. Он должен оставаться прозрачным при вставке в .pdf
